# WebApp - Webアプリ高速開発プロジェクト

## 重要：これはChrome拡張ではありません
HTMLファイルをダブルクリックで動くWebアプリを作ります。

## 開発哲学
- スピード最優先：10分で動くものを作る
- 単一ファイル：current/index.htmlに全部入れる
- ゼロ依存：CDN以外の外部依存なし

## スキル一覧
| コマンド | 役割 |
|---|---|
| /kickoff | 要件ヒアリング（5分） |
| /build | 実装（10分） |
| /snapshot | バージョン保存 |
| /iterate | 改善 |
| /media | カメラ・音声・画像の実装ガイド参照 |

## 技術スタック
- HTML5 / CSS3 / Vanilla JS
- CDN: cdn.jsdelivr.net / cdnjs.cloudflare.com
- データ保存: localStorage
